## 🧠 Descripción

**Reciper** es una aplicación web para buscar, crear y compartir recetas de cocina.
Desarrollada como proyecto final para un bootcamp Full-Stack, simulando un entorno profesional de desarrollo en equipo.

Permite a los usuarios registrarse y explorar recetas. Aquellos con rol de administrador pueden, además, crear, modificar o eliminar recetas e ingredientes de la web.

---

## ✨ Funcionalidades

- 🔐 Gestión de autenticación de usuarios con JWT y sistema de roles (admin/user)
- 📄 API REST para operaciones CRUD completas de recetas e ingredientes  
- 🔍 Búsqueda y filtro por categoría 
- 📱 Interfaz responsive 

---

## 🛠️ Tecnologías principales (MERN)

- **Frontend:** React + Tailwind CSS  
- **Backend:** Node.js + Express  
- **Base de datos:** MongoDB
